## 💻 Sobre o Projeto

O GrowTravel é uma página estática focada em agência de viagens, onde apliquei conceitos fundamentais de:
* Estruturação semântica com **HTML5**.
* Estilização avançada e responsividade com **CSS3**.
* Metodologia de organização de ficheiros para produção.
